# Melting Brain

A 2D physics puzzle game where the player draws lines to guide and protect a moving object so it reaches and stays in the goal zone.

## Game Mechanics
- line drawing with collision and rigid body.
- Ball is in freeze state until you draw a line and release the mouse
- 30 playable levels with menu pages.

## Objects
- Ball(Main object that should enter the glass)
- White slab()

## Goal
- Make the ball enter the glass by drawing shapes

## Control
- Mouse dragging
